# Cleanpro - Landing Page Jasa Cuci Sofa Probolinggo

Cleanpro adalah sebuah landing page profesional dan responsif untuk layanan jasa cuci sofa, springbed, karpet, dan jok mobil yang berlokasi di Probolinggo. Website ini didesain dengan gaya *modern, clean*, dan *premium* menggunakan skema warna Teal Material Design.

## 🚀 Fitur Utama

- **Desain Modern & Premium:** Menggunakan estetika desain bersih (*clean*) dengan warna utama Teal, bayangan halus (*subtle shadows*), dan layout yang rapi.
- **Responsif Sepenuhnya:** Tampilan optimal di perangkat *mobile*, tablet, dan *desktop*.
- **Smart Autohide Navbar:** Navbar otomatis menyembunyikan diri saat pengguna *scroll* ke bawah untuk memperluas area baca, dan muncul kembali saat *scroll* ke atas.
- **Carousel Harga Layanan:** Menampilkan paket harga mulai (Cuci Sofa, Cuci Springbed, Cuci Jok Mobil) dengan fitur *auto-scroll*, navigasi *dots*, dan dukungan gestur *swipe/drag*.
- **Carousel Testimoni Pelanggan:** Menampilkan ulasan pelanggan dalam format *carousel* interaktif.
- **Optimasi SEO Lokal:** Dilengkapi dengan struktur HTML semantik, *meta tags*, dan *schema markup* (JSON-LD) khusus *Local Business* untuk meningkatkan visibilitas di mesin pencari.
- **Offline Assets:** Semua *library* eksternal (Bootstrap dan Bootstrap Icons) disimpan secara lokal sehingga *loading* lebih cepat dan stabil tanpa bergantung pada koneksi internet eksternal.

## 💻 Tech Stack

- **HTML5:** Struktur semantik.
- **CSS3:** *Styling* khusus dan responsif (tanpa *framework utility* yang berat).
- **Vanilla JavaScript:** Logika interaktivitas untuk *carousel* dan *navbar* (tanpa jQuery).
- **Bootstrap 5 (Lokal):** *Grid system* dan komponen dasar.
- **Bootstrap Icons (Lokal):** Ikon profesional.

## 🛠️ Cara Menjalankan

Karena website ini adalah *static web*, Anda tidak memerlukan server khusus (seperti Node.js atau PHP) untuk menjalankannya.

1. **Unduh atau Clone** *repository* ini.
2. Buka folder proyek.
3. Klik dua kali pada file `index.html` untuk membukanya di *browser* (Google Chrome, Firefox, Safari, dll).
4. *Atau*, Anda bisa menggunakan ekstensi **Live Server** di VS Code untuk pengalaman *development* yang lebih baik dengan fitur *auto-reload*.
